### 开始解题

```sh
yarn start
```

跟随提示，输入题目 ID，标题，索引，难易度，将会自动建立对应文件夹并记录信息至 `package.json`。

结果检验注意事项，为满足多参数输入，传参用 `[ ]` 包裹。

### 解题完毕

```sh
yarn solve
```

输入对应题目 ID，将会通过自动提交带有题目信息的 `commit`。

如：`git commit -m "✅ solve palindrome-number"`
